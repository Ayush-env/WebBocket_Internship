2) If-Else<br>
if condition:<br>
  code to execute if condition is true<br>
else:<br>
  code to execute if condition is false<br><br>

a = 0<br>
if a>0<br>
  print("Num is pos")<br>
else:<br>
  print("Num is neg")<br><br>

3) Elif<br>

if condition1:<br>
  code to execute if condition is true<br>
elif condition 2:<br>
  code to execute if condition 2 is true<br>
else:<br>
  code to execute if both condition1 and condition 2 are false<br><br>

a = 0<br>
if a>0<br>
  print("Num is pos")<br>
elif a<0:<br>
  print("Num is neg")<br>
